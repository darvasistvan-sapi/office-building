
# Office Building – Webalkalmazás

## Szükséges előfeltételek

Az applikáció futtatásához szükséges a Node.js telepítése, amely tartalmazza az npm-et és az npx-et is.

- Node.js letöltése: [https://nodejs.org/](https://nodejs.org/)

## Az applikáció futtatása

1. Nyisson egy parancssort (Command Line/Terminal) a projekt gyökérkönyvtárában.
2. Futtassa az alábbi parancsot:

   ```sh
   npx serve app
   ```

3. A parancs futtatása után a terminálban megjelenik egy URL-cím, amelyen az alkalmazás elérhető lesz. Alapértelmezett cím: [http://localhost:3000](http://localhost:3000)
4. Nyissa meg ezt a címet a böngészőjében, hogy használni tudja az applikációt.
